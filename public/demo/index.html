How to Test INKLUSIV Features

  • Install the INKLUSIV extension and look for the floating widget in the bottom-right corner
  • Click the widget to expand accessibility controls
  • Try voice commands: "focus search", "type hello world", "scroll down", "click submit"
  • Enable gesture input and use hand gestures: point to scroll, peace sign to scroll up, OK to click
  • Test accessibility features: dyslexia font, text-to-speech, high contrast, dark mode
  • Hover over text with TTS enabled to hear it read aloud
  • Use the colorblind filters to see how they affect the page appearance

The Importance of Web Accessibility

Web accessibility ensures that websites, tools, and technologies are designed and developed so that people with disabilities can use them. More specifically, people can perceive, understand, navigate, and interact with the Web, and they can contribute to the Web.

Accessibility encompasses all disabilities that affect access to the Web, including auditory, cognitive, neurological, physical, speech, and visual disabilities. Accessibility also benefits people without disabilities, for example, people using mobile phones, smart watches, smart TVs, and other devices with small screens.

The Web is fundamentally designed to work for all people, whatever their hardware, software, language, location, or ability. When the Web meets this goal, it is accessible to people with a diverse range of hearing, movement, sight, and cognitive ability.

INKLUSIV Features in Detail

INKLUSIV provides a comprehensive suite of accessibility tools designed to make web browsing more inclusive. The extension includes voice recognition capabilities that allow users to navigate and interact with web pages using natural speech commands.

The gesture recognition feature uses advanced computer vision to track hand movements, enabling users to control their browser through intuitive gestures. This is particularly useful for users who may have difficulty using traditional input methods.

Visual accessibility features include high contrast modes, dark themes, dyslexia-friendly fonts, and color vision filters. These tools help users with various visual impairments or preferences to better perceive and interact with web content.
